# 🧠 الملخص النهائي الذكي (Smart NLP Revision)

## 📌 1. الأساسيات (Fundamentals)
* **NLP**: فهم ومعالجة اللغة البشرية.
* **Tokenization**: تقسيم النص ← Tokens.
* **Stemming vs Lemmatization**: القص (Stemming) مقابل الإرجاع للأصل الصحيح (Lemmatization).

## 📊 2. تمثيل النص (Text Representation)
* **Bag of Words (BoW)**: يعتمد على التكرار ويهمل الترتيب.
* **TF-IDF**: يوازن بين الأهمية والتكرار، مهم في التصنيف.
* **Word Embeddings**: تمثيل دلالي (Semantic) مثل *Word2Vec, GloVe*.

## 🤖 3. النماذج (Models)
* **تقليدية**: Naive Bayes, Logistic Regression.
* **عميقة**: RNN (تسلسلي)، Transformer (سريع ويعتمد على Attention).

## ⚡ 4. أهم المفاهيم الحديثة
* **Attention**: يحدد الكلمات المهمة في السياق.
* **Transformer**: يعتمد على المعالجة المتوازية (Parallel Processing).

## 🧩 5. النماذج الشهيرة
* **BERT**: (Masked Model) لفهم النص.
* **GPT**: (Autoregressive) لتوليد النص.

## 📏 6. التقييم (Evaluation)
* **Accuracy / F1**: للتصنيف.
* **BLEU**: للترجمة | **ROUGE**: للتلخيص.

## 🌍 7. تحديات العربية
* Morphology معقدة، غياب التشكيل، اللهجات، والتصاق الكلمات.
